# 🌍 Travel World Website

## 📖 Project Overview

Travel World is a modern and fully responsive travel website developed using **HTML5** and **CSS3**. The website helps users explore beautiful travel destinations, learn about the company, and contact the travel team through an easy-to-use contact form.

The project focuses on creating a clean, responsive, and user-friendly interface suitable for all screen sizes.


## Live Demo

🔗 [View Project](https://brilliant-biscotti-5dffb1.netlify.app)

- Explore Places Section
- Beautiful Destination Cards
